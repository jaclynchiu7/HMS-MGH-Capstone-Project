There are US state and county level map and covid-19 cases.
1. ST_CARTO.zip: US state shapefile
2. CO_CARTO.zip: US county shapefile
3. US_State_Case.csv: US stateCOVID-19 accumulated daily cases.
4. US_County_Case.csv: US county COVID-19 accumulated daily cases.

*Source: covid-19 cases are collected from https://github.com/CSSEGISandData/COVID-19
